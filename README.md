## Overview

整理 PDF 合并拆分、OCR、格式转换、电子书管理、扫描件归档和数学公式识别工具。

本仓库只保留与主题直接相关、入口稳定、说明清晰的资源。优先收录官方文档、主流开源仓库、长期可访问的产品页面和常用工具链。

## Recommended Path

1. 扫描件先 OCR，再归档。
2. 论文 PDF 用 Zotero 管理，办公 PDF 用 Stirling/PDF24。
3. 大批量转换优先 Pandoc 或脚本。

## Notes

- 敏感文件不应上传到未知在线工具。
- OCR 结果必须人工核对。
